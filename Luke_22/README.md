# Melkebar

**Descrition**:
> Etter en lang dag så gjør det seg godt med litt melk. Tok et raskt søk for å prøve å finne noen melkebarer jeg kunne besøke. Den eneste jeg fant var denne, som dessverre er permanent stengt. Men hvor lå den?

**Handout**: [melkebar](melkebar.png)

The text `BAYERISCHE` together with `MILCH-TRINKHALLE` points at `Bayerische Milchversorgung` [whitch is closesly connected to Nürnberg](https://pm20.zbw.eu/folder/co/0417xx/041738/about.de.html). This leads to the actual kiosk was in Nürnberg Hauptbahnhof, after some other guesses.
